#  WSN Intrusion Detection System using Machine Learning

A machine learning-based **Wireless Sensor Network (WSN) Intrusion Detection System** built using **Python, Streamlit, Scikit-learn, and XGBoost**.

This project helps detect malicious activities in network traffic by analyzing packet-related features and classifying them as **normal or attack traffic**.

##  Project Overview

Wireless Sensor Networks are highly vulnerable to cyber attacks due to limited resources and open communication channels.

This project uses machine learning models to identify suspicious behavior and improve network security.

The application is deployed using **Streamlit** and provides an interactive dashboard for predictions and visualization.

##  Features

* Real-time intrusion detection
* Interactive Streamlit dashboard
* Data visualization using Plotly
* Machine learning-based prediction
* Pre-trained model artifacts loading
* Attack vs normal traffic classification
* Deployed web app support
